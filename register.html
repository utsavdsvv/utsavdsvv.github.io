Register

Rules and Regulations

1. There are two kinds of events : individual and team events.
2. A player can participate in 3 indvidual events and 3 team events maximum.
3. The player has to individually register for all the individual events he/she wants to participate.
4. For team events, the captain of the team needs to register on the behalf of the whole team.
5. The schedule of events may change due any reasons.
6. The registered players will get notified for any further updates via email/mobile number.

Required details

For individual events:

1. Student ID number
2. Name
3. Course
4. Mobile number and email id

For team events:

1.Team Name
2. Name of Captain
3. Name of Vice-captain
4. Mobile number and email id
